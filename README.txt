KLS SameDay Office v15 — Accounts & Payments

INSTALL
1. Run KLS_SameDay_v15_Accounts_Payments.sql in Supabase SQL Editor.
2. Replace app.js, styles.css, index.html, manifest.json, sw.js, build.js and package.json in GitHub.
3. Commit summary: Upgrade to v15 Accounts and Payments
4. Push and wait for Vercel Ready.
5. Hard refresh and open Accounts & Payments.

FEATURES
- Invoice paid, part-paid, unpaid and overdue balances
- Record payment amount, date and method
- Outstanding and overdue totals
- Expense tracking
- Monthly income, costs and profit overview
- Existing v14 functionality retained
