KLS SameDay Office v9 — Driver App, Live Tracking & POD

INSTALL ORDER
1. In Supabase open SQL Editor > New query.
2. Open KLS_SameDay_v9_Tracking_POD_Upgrade.sql from this package, copy all, paste into the editor and click Run.
3. Replace the website files with: index.html, app.js, styles.css, manifest.json, sw.js, build.js and package.json.
4. Commit and push to GitHub. Wait for Vercel to show Ready.
5. Sign out, sign back in, then hard refresh.

NEW FEATURES
- Driver App page
- Driver status updates
- Phone GPS live location updates while Driver App remains open
- Private customer tracking links
- Delivery photo capture
- Signature pad
- Recipient name and delivery notes
- GPS and time evidence on POD
- POD storage in Supabase

IMPORTANT
Browser/PWA live tracking works while the Driver App is open. Reliable tracking while the phone is locked requires a later native iPhone/Android driver app.
