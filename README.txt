KLS SameDay Office v14 — Booking Calendar

INSTALL
1. Run KLS_SameDay_v14_Booking_Calendar.sql in Supabase SQL Editor.
2. Replace app.js, styles.css, index.html, manifest.json, sw.js, build.js and package.json in GitHub.
3. Keep your existing config.js or Vercel environment variables.
4. Commit summary: Upgrade to v14 Booking Calendar
5. Push and wait for Vercel Ready.
6. Hard refresh and open Booking Calendar.

FEATURES
- Monthly job calendar
- Drag jobs onto another date to reschedule
- Direct quick booking form
- Automatic low-workload driver allocation
- Recurring booking templates
- Generate one or all due recurring jobs
- Existing v13 functionality retained
